#前端项目

##项目依赖
element-ui
axios
lodash
moment

##项目命令
    npm/yarn 
    npm run dev   ------------------------------ 开发模式下启动项目
    npm run build ------------------------------ 生产模式下启动项目
    npm run dll   ------------------------------ dll打包依赖

##支持语法
    es5,es6,es7,es8 ES草案Stage 0， Stage 1， Stage 2， Stage 3
